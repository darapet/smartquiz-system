Groq AI — First Priority (up to 20 keys)

Quiz & challenge generation tries Groq first — blazing fast with a generous free tier. Falls back to Mistral if all Groq keys are busy. Get free keys at console.groq.com. Empty slots are ignored.

⚡ Priority order: Groq keys (slots 1–20) → Mistral keys (slots 1–20). Rate-limited keys cool down 62 s automatically, then rejoin the pool.

Library AI Keys — Dedicated Pool (10 slots)

These keys are used only for Library features (AI Explain, AI MCQ Practice) — completely separate from the main quiz AI pool. Falls back to main pool if all 10 are busy. Empty slots are ignored.

📚 How it works: Students using AI Explain and AI MCQ Practice on library PDFs use these 10 keys in round-robin rotation. If all are rate-limited, the system automatically falls back to the main Groq pool.
